# Security policy

## Reporting a vulnerability

If you discover a security vulnerability in this project, please open a [GitHub issue](https://github.com/cudeso/misp2sentinel/issues).

Please include:

- A description of the vulnerability
- Steps to reproduce

## Response

This is an open source project maintained in my free time. I will do my best to address reported vulnerabilities in the time available to me. There are no paid bug bounties.
